## üéØ How to Add Screenshots

### Using Git:
```bash
# From project root
cp /path/to/your/screenshot.png docs/screenshots/
git add docs/screenshots/
git commit -m "Add screenshot: [description]"
git push origin main
```

### Using GitHub Web:
1. Navigate to this folder on GitHub
2. Click "Add file" ‚Üí "Upload files"
3. Drag and drop your screenshots
4. Commit with descriptive message

## üìä Screenshot Guidelines

- **Format**: PNG (preferred) or JPG
- **Resolution**: Minimum 1280x720, preferably 1920x1080
- **Size**: Keep under 5 MB per file
- **Quality**: Clear, well-lit, no blur
- **Content**: Show actual functionality, no mocks

For detailed guidance, see [SCREENSHOTS_GUIDE.md](../SCREENSHOTS_GUIDE.md).
